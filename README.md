
# Team3 Functional Annotation


## Dependencies

- Infernal and Rfam

  Infernal may be obtained from bioconda:

  ```
  conda install -c bioconda infernal=1.1.2
  ```

  To download the Rfam database, use the `dependencies/prepare_Rfam.sh` script.

  ```
  cd dependencies
  ./prepare_Rfam.sh
  cd ..
  ```
